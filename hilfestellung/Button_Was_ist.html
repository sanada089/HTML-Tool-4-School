HTML

= Hypertext Markup Language
das heißt auf Deutsch: Auszeichnungssprache für Hypertext

HTML ermöglicht es eine Website zu gestalten. Hierbei kann über Tags bestimmt werden, wo auf der Seite etwas erscheinen soll.
Ein Tag besteht immer aus einem Anfang, einem Ende und einem Text dazwischen.
So leitet zum Beispiel die Zeichenfolge < h1 > eine Überschrift ein. Am Ende der Überschrift schreibt man dann < /h1 >
So sieht zum Beispiel < h1 > Hallo HTML < /h1 > in Html folgendermaßen aus:

Hallo HTML


Auf dieser Seite kannst du durch Blöcke ganz einfach verschiedene Tags kennenlernen und so verstehen, wie man Websites mit Hilfe von HTML gestalten kann.

Möchtest du noch mehr über HTML erfahren findest du im Internet zahlreiche Quellen. Zum Beispiel: wikipedia
oder W3Schools.com!
