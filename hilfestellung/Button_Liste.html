Listen

Sie werden verwendet, um formatierte Informationslisten zu erstellen. In HTML gibt es
  • geordnete Listen: < ol> Listeninhalt < /ol>
    (engl. ordered list)
  • ungeordnete Listen: < ul> Listeninhalt < /ul>
    (engl. unordered list)

Die Elemente (also der Inhalt) der Listen werden zwischen den Tags
< li> Listeninhalt < /li> platziert.

Der Aufbau von ungeordneten Listen ist identisch zu den geordneten Listen, nur das entsprechende Listen-Tag ändert sich. Mit dem Attribut type kann der Aufzählungsstil verändert werden.

Folgende Aufzählungszeichen sind möglich:

  • circle (= Kreis)
  • square (= Quadrat)
  • disc (= ausgemalter Kreis)
