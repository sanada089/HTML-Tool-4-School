Tabellen

In Tabellen können Texte oder Daten geordnet zusammengestellt werden.
Inhalte werden in
  • Zeilen ▸ waagerecht
  • und Spalten ▸ senkrecht
gegliedert. HTML gibt den Aufbau der Gitterstruktur vor.
Für den Aufbau der Gitterstruktur sind mehrere Elemente notwendig:
  • < table> leitet eine Tabelle ein (table = Tabelle), < /table> schließt die Tabelle
  • < th> Kopfzelle < /th> (table header = Tabellenkopf)
    ▸ Der Tabellenkopf ist sozusagen die Überschrift der Tabellenspalte.
  • < tr> table row = Tabellenzeile
    ▸ leitet eine Tabellenzeile ein, die Datenzellen folgen
    ▸ < /tr> ist das schließende Tag der Zeile
  • < td> Datenzelle < /td> (tabledata = Tabellendaten)
    ▸ Hier wird der Inhalt der Tabelle notiert.
